## data 文件结构

data 文件（如 `627531647`）的 JSON 结构：

```json
{
  "answerIDMap": {},
  "fileNameData": [...],
  "newStruct": true,
  "paperId": "760241",
  "sectionData": [
    {
      "caption": "模仿朗读",
      "category": "read_chapter",
      "newStruct": true,
      "paperId": "760241",
      "sectionItemData": [
        {
          "category": "read_chapter",
          "childPaperBean": {...},
          "fileName": "7eccfa00642f290a9e2ae159309284b4",  // 资源文件夹名（顶层！）
          "mainItem": false,                              // false=具体题目，true=汇总入口
          "itemCount": 1,
          "maxScore": 6.0,
          ...
        }
      ]
    }
  ]
}
```

**关键发现**：
- `fileName` 在 `sectionItemData` **顶层**（不是 `childPaperBean.fileName`）
- `mainItem=true` 是汇总入口，应该**跳过**
- `mainItem=false` 是具体题目，需要处理
- 同一个 `fileName` 可能被多个 `sectionItemData` 引用，需要**去重**

## 正确的读取流程

### Step 1: 扫描 data 目录，筛选有效文件 (>50KB)

```kotlin
val dataFiles = reader.listFiles(dataDirPath)
    .filter { !it.isDirectory && it.size > MIN_FILE_SIZE }
```

### Step 2: 解析 data 文件获取试卷结构

```kotlin
val json = JSONObject(content)
val sectionData = json.optJSONArray("sectionData")
```

### Step 3: 遍历 sectionData，处理每个 sectionItemData

```kotlin
val processedFileNames = mutableSetOf<String>()

for (section in sectionData) {
    val sectionItemData = section.optJSONArray("sectionItemData")
    
    for (item in sectionItemData) {
        val mainItem = item.optBoolean("mainItem", false)
        if (mainItem) continue  // 跳过汇总入口
        
        val fileName = item.optString("fileName", "")
        if (fileName.isEmpty()) continue
        if (fileName in processedFileNames) continue  // 跳过重复
        processedFileNames.add(fileName)
        
        // 读取答案
        val answerContent = readContentJson(reader, fileName)
    }
}
```

## 题目重复问题解决方案

**问题原因**：
1. `mainItem=true` 的 sectionItemData 是汇总入口，不是具体题目
2. 同一个 `fileName` 被多个 sectionItemData 引用

**解决方案**：
```kotlin
val processedFileNames = mutableSetOf<String>()

for (item in sectionItemData) {
    val mainItem = item.optBoolean("mainItem", false)
    if (mainItem) continue  // 跳过 mainItem=true
    
    val fileName = item.optString("fileName", "")
    if (fileName in processedFileNames) continue  // 跳过重复
    processedFileNames.add(fileName)
    
    // 处理答案
}
```

## 数据模型（重构后）

```kotlin
data class Paper(
    val paperId: Long,
    val dataFileName: String,
    val fileSize: Long,
    val sections: List<Section>  // 按 section 组织
)

data class Section(
    val caption: String,           // 如"模仿朗读"
    val category: String,           // 如"read_chapter"
    val typeName: String,           // 中文类型名称
    val questions: List<Question>, // 该 section 下的所有题目
    val originalContent: String?    // 原文内容
)

data class Question(
    val order: Int,                // 全局题号
    val sectionOrder: Int,         // 所在 section 的题号
    val sectionCaption: String,   // 所属大题名称
    val typeName: String,          // 题型名称
    val questionText: String,      // 题目文本
    val answers: List<String>,     // 标准答案列表
    val originalText: String?      // 原文
)
```

3. **追踪已处理的 fileName** 避免重复
4. **答案在 info.question[].std[].value**
5. **原文在 info.value**
